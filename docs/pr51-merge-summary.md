## 🎯 功能实现

### 1. benchmark_meditation.py - 冥思性能基准测试

**测试规模：**
| 规模 | 节点数 | 用途 |
|------|--------|------|
| small | 100 节点 | 快速测试 |
| medium | 500 节点 | 常规测试 |
| large | 1000 节点 | 负载测试 |
| xlarge | 5000 节点 | 压力测试 |

**性能指标：**
- 冥思时长（秒）
- 节点处理速度（节点/分钟）
- LLM 调用次数（估算）
- 总成本（估算，美元）
- 熵减效果（%）

### 2. benchmark_search.py - 检索性能基准测试

**测试查询：** 10 个标准查询（人工智能、机器学习、深度学习等）

**性能指标：**
- 平均响应时间（毫秒）
- P95 响应时间（毫秒）
- P99 响应时间（毫秒）
- 最小/最大响应时间

### 3. 使用方法

```bash
# 冥思基准测试
python scripts/benchmark_meditation.py --scale small --dry-run

# 检索基准测试
python scripts/benchmark_search.py --iterations 10 --dry-run

# 保存结果
python scripts/benchmark_meditation.py --scale medium --output-dir ./results
python scripts/benchmark_search.py --iterations 20 --output-dir ./results
```
