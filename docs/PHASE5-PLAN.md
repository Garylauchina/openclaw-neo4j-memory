## 🎯 Phase 5 核心目标

**主题：** 从"可靠记忆系统"到"智能记忆体"

**核心升级：**
1. **混合检索** - 图遍历 + 向量相似度双路召回
2. **记忆分层** - 临时上下文、稳定事实、偏好习惯、任务状态
3. **状态持久化** - 冥思等异步任务可恢复（幂等性设计）
4. **性能基准** - 量化不同规模下的资源消耗

### Issue #44: 记忆分层设计（Phase 5 P0）

**状态：** 📋 待创建  
**难度：** 🟡 中等  
**工时：** 4-6 小时

**任务内容：**
1. 定义记忆分层模型
2. 实现分层存储逻辑

**记忆分层模型：**
| 层级 | 类型 | 生命周期 | 例子 |
|------|------|----------|------|
| L1 | 临时上下文 | 会话级 | 当前对话内容 |
| L2 | 稳定事实 | 永久 | 用户姓名、职业 |
| L3 | 偏好习惯 | 长期 | 回答风格、时区 |
| L4 | 任务状态 | 任务周期 | 待办事项、进度 |
| L5 | 推理过程 | 可压缩 | 决策逻辑、因果链 |
