# Issue Layer Mapping Draft

## Purpose

This draft maps the current issue landscape into the new four-stage project mainline:

- memory
- experience
- knowledge crystal
- Agent-to-Agent transfer

This is not a final taxonomy.
It is a working reorganization layer for discussion and later refactor decisions.

---

## Four-Layer Mainline

### Layer 1. Memory
Retention of source material, provenance, graph persistence, grounding, contamination control.

### Layer 2. Experience
Reflection, meditation, candidate generation, hypothesis/stable distinctions, updateable processed cognition.

### Layer 3. Knowledge Crystal
Fold / unfold, reasoning-path crystal, convergence fidelity, calibration-aware crystal design, content core / adaptation layer.

### Layer 4. Agent-to-Agent Transfer
Cross-agent reconstruction, prompt-fed crystal injection, target-side adaptation rebuild, transfer fidelity, future implementation regeneration.
