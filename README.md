# openclaw-neo4j-memory

A research and implementation project for turning retained memory into transferable machine-native cognitive structure.

## Current Mainline

The project is now better understood as:

## memory -> experience -> knowledge crystal -> Agent-to-Agent transfer

Rather than only as a graph memory system.

---

## What This Project Is

`openclaw-neo4j-memory` started as a Neo4j-backed memory system for OpenClaw agents.

It is now evolving into a broader pipeline with four connected stages:

### 1. Memory
Retain source material, provenance, and local evidence.

### 2. Experience
Process retained material into usable cognitive structure through reflection, meditation, candidate generation, and feedback.

### 3. Knowledge Crystal
Compress structured experience into machine-native growth seeds that can be unfolded with controlled reasoning.

### 4. Agent-to-Agent Transfer
Transfer structured cognitive cores across agents, allowing target-side calibration and adaptation rebuild.
